🔬 Advanced Parameters

MR0: Li–Stephens; MR1: estimates one hotspot; MR2: one or two user-specified hotspot intervals; MR3: constant (estimated); MR4: constant (fixed, requires -R).
PHASE v2.1 allows n=1 hotspot with unknown position. The UI will pass -MR1 1.
Example: -MR2 1 4200 5400 or -MR2 2 a1 b1 a2 b2.
PHASE uses up to 100 individuals (randomly chosen) for recombination rate estimation by default. Computation increases quadratically with more individuals. Override only if needed for accuracy.
Stepwise: microsatellites change by ±1 repeat (default); PIM: mutations to any allele (tri-allelic SNPs, HLA); δ-file: custom per-locus mutation parameters (0=stepwise, 1=PIM).
